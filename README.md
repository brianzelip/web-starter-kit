# Web starter kit

This is a [gulp](http://gulpjs.com)-based package for jump starting web pages. It implements the following build tools:
- Local web server with livereload
- Sass compiler
- CSS autoprefixer
- CSS minifier
- Markdown compiler

Other components include:
- google web fonts
    + merriweather
    + ?
